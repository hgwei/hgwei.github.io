Markdown常用语法

Markdown 是一种轻量级标记语言，很适合写技术文档。这里整理了一些Markdown的常用语法，方便参考使用。

# 目录

在文档中使用如下指令，即可生成目录，一般用在文章最顶部。
[[toc]]

# 标题

格式：# + 空格 + 文字 + 回车 （其中 # 的数量对应了几级标题）
提示：#的数量对应了几级标题，在`Typora`中，可以使用快捷键ctrl+数字，例如：ctrl+1表示一级标题。

# 这是一级标题
## 这是二级标题，二级标题底下有横线
### 这是三级标题
#### 这是四级标题
##### 这是五级标题
###### 这是六级标题

# 列表

1、无序列表：使用-+*任意一个字符 + 空格 +内容
- 列表内容1
+ 列表内容2
* 列表内容3

2、有序列表：使用数字 + 点 + 空格 + 内容
1. 列表内容1
2. 列表内容2
3. 列表内容3

3、任务列表，- + 空格 + [x] + 空格 + 内容
- [x] 选项一（默认选中）
- [ ] 选项二（默认未选中，大括号里有空格）

4、列表嵌套：第二行缩进两个空格就能嵌套
- 一级列表内容1
  1. 二级列表内容11
  2. 二级列表内容12
- 一级列表内容2
  1. 二级列表内容21
  2. 二级列表内容22

# 表格

-使用 | 来分割表头内容，新增一行使用tab键
-第二行代表文字样式，-表示文字默认居左；:-:表示文字居中；-:表示文字居右
如下：

姓名|性别|年龄
-|:-:|-:
内容默认居左|内容居中|内容居右
张三|男|20
李四|女|22
王五|男|23
