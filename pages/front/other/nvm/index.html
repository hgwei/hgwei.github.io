nvm管理nodejs版本工具

提示

nvm 一个nodejs版本管理工具，可以管理多个版本的nodejs，让你电脑可以同时安装多个不同版本的nodejs。

注意：安装nvm前，需提前卸载本机安装的nodejs。

# 一、下载安装nvm

1、安装包下载地址：nvm github下载地址 (opens new window)

2、双击nvm-setup.exe安装，选择nvm安装路径。

3、下一步，选择nodejs安装路径（建议在nvm安装路径内建一个nodejs目录，方便管理）

4、安装完成，打开dos，输入nvm -v 验证是否安装成功

# 二、配置国内镜像

打开安装路径：D:\software\nvm，中的settings.txt文件，添加如下配置：

node_mirror: https://npm.taobao.org/mirrors/node/
npm_mirror: https://npm.taobao.org/mirrors/npm/

# 三、使用nvm管理nodejs

# 查看可安装的nodejs版本
nvm list available
1
# 安装指定的nodejs版本
nvm install 16.15.0
1
# 查看已安装的nodejs版本
nvm list
1
# 使用指定的nodejs版本
nvm use 18.18.0
1
# 卸载指定nodejs版本
nvm uninstall 18.18.0
1
